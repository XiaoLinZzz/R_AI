## Dataset Features

The datasets contain various data types including:

- 📝 Text
- 📅 Dates
- 🔢 Integers
- 💫 Floats
- ✅ Booleans
- 📊 Categories
- 🔄 Complex numbers

These diverse data types make our datasets particularly well-suited for testing data type inference and conversion functions.
